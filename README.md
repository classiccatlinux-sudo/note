# note
a simple CLI sticky note program made in python3

# how to use note 1.1

1. install python3

2. run the raw .py file. or allow the run.sh to run as a program then execute it.

3. enter your note

4. any key on the keyboard to stop (or if that does not work use ctrl + c to stop)
